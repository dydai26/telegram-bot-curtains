# Телеграм Бот для продажу Тюлей та Штор 🪟

Бот для автоматизації продажів з вбудованим калькулятором вартості та
AI-консультантом.

## 🚀 Функціонал

- **Каталог:** Перегляд товарів за категоріями (Тюль, Штори).
- **Калькулятор:** Розрахунок вартості за розмірами (ширина × висота).
- **Оформлення замовлень:** Збереження даних у Supabase.
- **AI-Консультант:** Підтримка клієнтів за допомогою OpenClaw або OpenAI.
- **Оплата:** Інтеграція зі Stripe/LiqPay (у процесі).

## 🛠 Технології

- Node.js + TypeScript
- Telegraf (Telegram Bot API)
- Supabase (Database & Auth)
- AI: OpenClaw / OpenAI

## 📦 Встановлення

1. Клонуйте репозиторій.
2. Встановіть залежності:
   ```bash
   npm install
   ```
3. Створіть файл `.env` на основі `.env.example` та заповніть токени.
4. Налаштуйте базу даних у Supabase за допомогою `schema.sql`.
5. Запустіть бота:
   ```bash
   npm run dev
   ```

## 🗄 Структура бази (Supabase)

- `categories`: Категорії товарів.
- `products`: Товари з ціною за метр.
- `orders`: Замовлення клієнтів.
- `bot_users`: Користувачі бота.

## 🤖 Налаштування AI

Для роботи консультанта змініть `OPENAI_API_KEY` або вкажіть `OPENCLAW_URL` у
файлі `.env`. Logic AI знаходиться у `src/ai.ts`.
